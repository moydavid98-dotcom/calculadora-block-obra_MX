# Calculadora de Block – Obra MX

App web desarrollada con Streamlit para cálculo rápido de blocks, mortero, costos y pesos,
alineada a práctica común de obra en México.

## Uso
1. Ingresar dimensiones del block
2. Ingresar datos del muro
3. Presionar CALCULAR
4. Exportar PDF

Desarrollada para uso en obra desde celular.
